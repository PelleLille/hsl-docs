Halon Scripting Language
========================

Halon's scripting language is a domain-specific language (`DSL <http://en.wikipedia.org/wiki/Domain-specific_language>`_) for Halon's `email platform <http://halon.io/>`_. It's used to configure the behavior of email processing in a scriptable fashion.
